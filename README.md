# Logo Carousel per Notion

Apri `index.html` nel browser per un test locale.
Poi pubblica tutto su GitHub Pages.

## File da toccare al volo
Apri `style.css` e modifica solo le variabili in alto:

- `--logo-height`
- `--section-height`
- `--scroll-duration`
- `--gap`

## Come aggiungere o togliere loghi
Apri `index.html` e modifica i tag `<img>` dentro i due blocchi `.logo-strip__group`.

Importante: i due gruppi devono essere identici, altrimenti il loop non sarà fluido.
